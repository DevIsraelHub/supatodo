- Server Actions
- useFormState

- Supabase Database
- Supabase Auth
- Auth and oAuth example (GitHub)

### Learning Points

Hopefully, from this project and the video, you can learn:

- NextJS 14 Server Actions (and drawbacks)
- Supabase Auth and Database
- useFormState
- useOptimistic
